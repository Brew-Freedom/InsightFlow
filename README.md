# InsightFlow

A comprehensive marketing analytics platform combining frontend and backend services to deliver actionable business insights.

## Project Structure

```
InsightFlow/
├── frontend/          # React-based user interface
├── backend/           # Node.js/Express API server
├── data/              # Sample datasets
├── docs/              # Project documentation
└── README.md
```

## Getting Started

### Prerequisites
- Node.js (v14+)
- npm or yarn
- Git

### Installation

#### Frontend Setup
```bash
cd frontend
npm install
npm start
```

#### Backend Setup
```bash
cd backend
npm install
npm start
```
